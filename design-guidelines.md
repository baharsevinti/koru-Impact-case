# Koru Impact - Tasarım Rehberi ve Wireframe Açıklamaları

## 🎨 Tasarım Sistemi

### Renk Paleti
```
Primary Colors:
- Deep Blue: #1a365d (Ana marka rengi - güven ve profesyonellik)
- Forest Green: #2d5016 (Sosyal bilimler ve sürdürülebilirlik)
- Warm Gold: #d69e2e (Vurgu rengi - başarı ve değer)

Secondary Colors:
- Light Blue: #e6fffa (Arka plan vurguları)
- Soft Gray: #718096 (Metin ve ikincil elementler)
- Pure White: #ffffff (Ana arka plan)
- Dark Gray: #2d3748 (Koyu metin)
```

### Tipografi
```
Headings:
- Font: Inter, sans-serif
- H1: 48px, Bold (700)
- H2: 36px, Semi-bold (600)
- H3: 24px, Medium (500)
- H4: 20px, Medium (500)

Body Text:
- Font: Inter, sans-serif
- Size: 16px, Regular (400)
- Line Height: 1.6
- Color: #2d3748

CTA Buttons:
- Font: Inter, sans-serif
- Size: 18px, Semi-bold (600)
```

### Spacing System
```
8px Grid System:
- XS: 8px
- S: 16px
- M: 24px
- L: 32px
- XL: 48px
- XXL: 64px
- XXXL: 96px
```

## 🎯 UI/UX Prensipleri

### 1. Görsel Hiyerarşi
- **Hero Section:** En büyük başlık ve ana CTA
- **Section Headers:** Orta boyut başlıklar
- **Body Content:** Okunabilir metin boyutu
- **Secondary Info:** Küçük metin boyutu

### 2. Beyaz Alan Kullanımı
- Section'lar arası: 64px boşluk
- Element'ler arası: 24px boşluk
- İçerik kenar boşlukları: 32px

### 3. Responsive Tasarım
```
Breakpoints:
- Mobile: 320px - 768px
- Tablet: 768px - 1024px
- Desktop: 1024px+
```

### 4. Etkileşim Tasarımı
- **Hover Effects:** Butonlar ve linkler için
- **Loading States:** Form gönderimi sırasında
- **Success/Error States:** Form validasyonu için
- **Smooth Transitions:** Sayfa geçişleri için

## 🔧 Teknik Gereksinimler

### Frontend Framework
- **React.js** veya **Next.js** (SEO için)
- **TypeScript** (Tip güvenliği)
- **Tailwind CSS** (Hızlı geliştirme)

### Performans Hedefleri
- **Lighthouse Score:** 90+ (Tüm kategorilerde)
- **First Contentful Paint:** < 1.5s
- **Largest Contentful Paint:** < 2.5s
- **Cumulative Layout Shift:** < 0.1

### Erişilebilirlik
- **WCAG 2.1 AA** uyumluluğu
- **Keyboard Navigation** desteği
- **Screen Reader** uyumluluğu
- **Color Contrast** oranları
